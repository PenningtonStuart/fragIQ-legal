# Screenshots Directory

Place app screenshots here for the GitHub Pages site and Play Store listing.

## Required Screenshots (minimum 2)

1. `frags-list.png` - Main frags list showing the collection
2. `frag-detail.png` - Frag detail screen with photos

## Recommended Additional Screenshots

3. `add-frag.png` - Add/edit frag form
4. `tanks.png` - Tank management screen
5. `dashboard.png` - Dashboard with stats
6. `events.png` - Events/calendar view
7. `settings.png` - Settings screen
8. `photo-gallery.png` - Photo gallery view

## Specifications

- **Dimensions**: 1080x1920 pixels (9:16 portrait) or 1920x1080 (16:9 landscape)
- **Format**: PNG or JPEG
- **Max Size**: 8 MB per image
- **Min Count**: 2 for phone category
- **Max Count**: 8 for phone category

## Screenshot Sources

Screenshots can come from:
1. **Mockup HTML files** - Located in `mocks/designs/` (for pre-launch)
2. **Actual device captures** - Real app running on device
3. **CI/CD Pipeline** - Automated screenshot generation via Playwright
