# FragIQ

The official website for FragIQ — a coral frag collection management app for reef enthusiasts.

## About

This repository hosts the public-facing website for FragIQ, deployed via GitHub Pages. It includes:

- **Landing Page** — App overview and features
- **Privacy Policy** — How user data is collected and handled
- **Terms of Service** — Usage terms for the app
- **Account Deletion** — Instructions for requesting data deletion

## Structure

```
├── index.html           # Landing page
├── privacy.html         # Privacy Policy
├── terms.html           # Terms of Service
├── delete-account.html  # Account deletion info
├── css/
│   └── style.css        # Shared styles
└── assets/              # Images and screenshots
```
